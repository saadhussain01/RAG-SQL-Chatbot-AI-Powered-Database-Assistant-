# 📊 RAG SQL Chatbot (AI-Powered Database Assistant)

An intelligent chatbot that converts natural language questions into
secure SQL queries using RAG and executes them on a MySQL database.

## 🚀 Features

-   Natural language to SQL
-   RAG-based schema retrieval
-   Safe query execution
-   Streamlit UI
-   Image & URL support

## 📁 Project Structure

RAG-SQL-Chatbot/ ├── app.py ├── schema.txt ├── langchain_rag/ ├── utils/
├── db/ ├── requirements.txt └── README.md

## ⚙️ Tech Stack

Python, Streamlit, LangGraph, LangChain, FAISS, MySQL,
HuggingFace/Gemini/OpenAI

## ▶️ Run

streamlit run app.py
